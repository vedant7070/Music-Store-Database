. Music Store Database
   . Project Overview
      This project is a SQL-based Music Store Database developed using MySQL.
      It is designed to manage customer, artist, album, track, invoice, and employee data.

. Technologies Used
  - SQL
  - MySQL
  - MySQL Workbench

. Features
   - Database Design
   - Table Relationships
   - SQL Queries
   - Joins
   - Aggregate Functions
   - GROUP BY
   - ORDER BY
   - Subqueries

. Database Tables
   - Employee
   - Customer
   - Invoice
   - Invoice_Line
   - Artist
   - Album
   - Track
   - Genre
   - Playlist
   - Playlist_Track

 . Learning Outcomes
     - Relational Database Design
     - Writing SQL Queries
     - Data Analysis using SQL
     - Database Management
